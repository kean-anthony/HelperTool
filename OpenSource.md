# Commands Reference — Helper Tool

---

## Install Dependencies

```bash
npm install
```

Installs all project dependencies.

---

## Install Electron

```bash
npm install electron
```

Installs Electron runtime for the desktop app.

---

## Install Electron Builder (Dev Dependency)

```bash
npm install --save-dev electron-builder
```

Installs Electron Builder for packaging the app.

---

## Run App (Development)

```bash
# Old versions run command V1 - V10
npx electron main.js

# New version run command V11+
npm start
```

Starts the Electron app in development mode.

---

## Build Windows Executable

```bash
npm run build:win
```

Builds the Windows `.exe` installer.

---

## Open DevTools (Electron)

```
Ctrl + Shift + I
```

Opens developer tools inside Electron.

---

## Open Project in VS Code

```bash
code .
```

Opens the project in Visual Studio Code.
